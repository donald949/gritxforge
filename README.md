# GRIT x FORGE PayPlan Calculator

A comprehensive commission and compensation calculator for GRIT sales representatives.

## Features

- **Personal Commission Calculator** - Calculate upfront and backend commission based on revenue and payscale
- **Recruiter Override Calculator** - Track recruiting override earnings for team growth
- **Management Override Calculator** - Calculate management override based on team performance
- **Profit Share Calculator** - Determine profit sharing based on team revenue
- **Step Bonus Calculator** - Track step bonus qualifications and earnings
- **Total Compensation Summary** - View complete compensation breakdown with PDF export

## Files

- `login.html` - User authentication page
- `index.html` - Main calculator application
- `admin.html` - Admin portal for user management
- `grit x forge logo.png` - GRIT x FORGE logo

## Usage

1. Open `login.html` in a web browser
2. Login with your credentials (default admin: donald@forgemarketing.us / Alphateam1!)
3. Navigate through different calculator tabs to calculate compensation
4. Export results as PDF reports

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- jsPDF for PDF generation
- LocalStorage for session management

## Local Storage

The application uses browser LocalStorage to store:
- User accounts (`gritUsers`)
- Session data (`gritSession`)

## Demo

Simply open `login.html` in any modern web browser to start using the calculator.
